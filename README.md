# Presentia – Smart Attendance Management System

## 📌 Problem Statement

Traditional attendance systems are slow, error-prone, and difficult to manage. Manual attendance consumes valuable class time and makes report generation challenging.

Presentia is a modern attendance management platform that digitizes attendance tracking using QR-based verification, centralized record management, analytics dashboards, and role-based access control.

---

## 🚀 Features

### Admin

* Manage Students
* Manage Faculty
* Manage Subjects
* Manage Attendance Sessions
* View Attendance Records
* Access Reports & Analytics
* Monitor Institution-wide Attendance

### Faculty

* View Students
* Manage Sessions
* Mark Attendance using QR
* View Attendance Reports

### Student

* View Personal Attendance
* Access Attendance Reports
* Participate in QR Attendance Sessions

### Attendance Features

* QR-Based Attendance
* Session Management
* Attendance History
* Real-Time Dashboard
* Role-Based Access Control

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yaswanthkumar-07/attendance-management-system.git
cd attendance-management-system
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Environment Variables

Create `.env` file in backend:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret
```
