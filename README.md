# Presentia – Smart Attendance Management System

## 📌 Overview

Presentia is a modern Attendance Management System developed using the MERN Stack (MongoDB, Express.js, React, Node.js). The platform digitizes attendance management through centralized administration, QR-based attendance verification, role-based access control, and real-time reporting.

The system is designed to help educational institutions manage students, faculty, subjects, attendance sessions, and attendance records from a single platform.

---

# 🎯 Problem Statement

Traditional attendance systems are time-consuming, difficult to manage, and prone to human errors. Faculty members spend valuable class time recording attendance manually, while administrators face challenges in maintaining and analyzing attendance records.

Presentia addresses these challenges by providing a centralized web platform that automates attendance management and improves accessibility, transparency, and reporting.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yaswanthkumar-07/attendance-management-system.git

cd attendance-management-system
```

---

## Backend Setup

```bash
cd backend

npm install

npm start
```

---

## Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

---

## Environment Variables

Create a `.env` file inside the backend directory.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```
